SiYuan​

Obsidian​

安装​

只需要前往官网下载对应的软件并安装即可, 在安装完成后, 先进行一些简易的配置 前往核心插件, 添加 漫游笔记功能 以及 斜杆命令 两个功能, 这样便于后续使用

插件配置​

  • Easy Typing : 输入时格式化
  • excalidraw : 画图插件, 但是我没有安装
  • Editor Syntax Highlight : 代码高亮
  • Advanced Tables : 优化 OB 的表格输入
  • Obsidian 样式：多彩的 tag 样式
  • Thino
  • VSCode Editor : 提供查看和编辑各种代码文件和代码块的功能
  • Link Favicons : 为笔记中的网页链接增加网站图标
  • Auto Link Title : 自动为网页地址增加链接名
  • Auto Card Link : 将网页链接自动转化为卡片样式
  • Banners : 添加头图
  • Iconize : 文件图标
  • Admonition : 高亮块
