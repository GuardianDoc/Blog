# VSCODE
[VSCODE](https://code.visualstudio.com/) 微软推出的开源的免费编辑器, 功能全, 可扩展性高, 非常适合进行轻量型的开发, 但是对于 Java 这类项目支持度不够
## 插件推荐
- Project Manager : 它支持将任何文件夹或工作区另存为项目，支持在同一窗口或新窗口中打开项目等
- CSS Navigation : 通过点击类名、id、标签名即可跳转到具体样式定义的插件 [可选]
